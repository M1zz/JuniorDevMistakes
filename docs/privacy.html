개인정보처리방침

최종 업데이트: 2026년 2월

실수 100 앱(이하 "앱")은 사용자의 개인정보를 소중히 여깁니다. 본 방침은 앱이 수집하는 정보와 사용 방식을 설명합니다.

수집하는 정보

앱은 다음과 같은 데이터를 사용자 기기에 로컬로 저장합니다:

  • 체크리스트 완료 현황 (어떤 실수 항목을 체크했는지)
  • 회고 일지 내용 (사용자가 직접 작성한 답변)
  • 북마크 목록
  • 앱 구독 상태

이 데이터는 사용자의 Apple ID에 연결된 iCloud (CloudKit)를 통해 동일한 Apple ID로 로그인된 기기 간에 동기화될 수 있습니다.

수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처, 카메라, 마이크 등 기기 접근 권한
  • 사용 분석 데이터 또는 광고 추적

제3자 공유

앱은 수집한 어떠한 데이터도 제3자에게 판매하거나 공유하지 않습니다.

iCloud 동기화

iCloud 동기화를 사용하는 경우, 데이터는 Apple의 CloudKit 인프라를 통해 전송 및 저장됩니다. Apple의 개인정보처리방침이 해당 데이터 처리에 적용됩니다. Apple 개인정보처리방침: apple.com/legal/privacy

인앱 구매

구독 및 인앱 구매는 Apple App Store를 통해 처리됩니다. 결제 정보는 Apple이 처리하며 앱 개발자는 접근할 수 없습니다.

데이터 삭제

앱 내 설정 → "모든 데이터 초기화"를 통해 앱에 저장된 모든 데이터를 삭제할 수 있습니다. iCloud에 동기화된 데이터를 삭제하려면 iOS 설정 → Apple ID → iCloud → 앱 관리에서 삭제할 수 있습니다.

아동 개인정보

앱은 만 14세 미만 아동으로부터 의도적으로 개인정보를 수집하지 않습니다.

방침 변경

개인정보처리방침이 변경될 경우 이 페이지에 업데이트됩니다. 중요한 변경 사항은 앱 업데이트를 통해 안내됩니다.

문의

개인정보 관련 문의사항이 있으시면 App Store 개발자 페이지의 연락처를 통해 문의해 주세요.
